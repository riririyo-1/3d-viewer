# Studio View サイトマップ

```
/トップ画面
  - ヒーローセクション
    - トップバー
      - Manage Assets
        - open collection
        - import assets
      - ロゴ＋アプリ名称
      - 履歴
        - Recently Opened
    - アカウントアイコン
      - アカウント名+プラン
      - アカウント設定
      - 言語設定
      - アプリ設定
      - ログアウト
    - メインビジュアル
    - 機能カード
      - My Collection → /collection
      - Conversion → /conversion

/collection
  - インポートボタン
    - 対応形式
      - .obj  (Wavefront OBJ)
      - .glb  (glTF Binary)
      - .gltf (glTF JSON)

  - アセット一覧
    - カード表示 → /viewer
      - プレビュー (ない場合はBoxアイコン)
      - ファイル名
      - 拡張子バッジ (obj/glb/gltf)
      - 削除ボタン

/viewer
  - 3D キャンバス
    - 右クリック：カメラ移動、左クリック：カメラ回転、スクロール：カメラズーム
  - コントロールパネル
    - WIREFRAME
      - ワイヤーフレーム表示 ON/OFF
    - GRID
      - グリッド表示 ON/OFF
    - SPIN
      - 自動回転 ON/OFF

/conversion
  - アップロード画面
    - obj2glb
      - ファイル選択
      - 削除ボタン
      - 変換ボタン
    - obj2gltf
      - ファイル選択
      - 削除ボタン
      - 変換ボタン

/login
  - ログインフォーム
    - メールアドレス
    - パスワード
    - ログインボタン
    - Googleでログイン(未実装)

/settings
  - アカウント管理
    - プラン
    - ストレージ
  - 言語設定

```
